
# Rational Numbers Calculator  
  
## Описание проекта  
  
Этот проект представляет собой эмулятор калькулятора, который работает с дробями  в различных системах счисления. Калькулятор поддерживает арифметические операции над рациональными числами и выводит результат в заданной системе счисления. Результат каждой операции должен быть выведен в заданных системах счисления и представлен в каноническом виде (несократимая дробь).  
  
## Возможности калькулятора  
  
Калькулятор поддерживает следующие операции:  
  
- **+**: Сложение  
- **-**: Вычитание  
- **\***: Умножение  
- **/**: Деление  
- **finish**: Завершение работы калькулятора  
  
## Формат ввода  
 В данной задаче калькулятор работает с командами в формате:  
 - `<операция>` `<число>`
	- Число: `<система_счисления>:<пробельный_символ>[<знак>]<целая_часть_числитель>[/<натуральная_часть_знаменатель>]`  
	- Знак: 
		- **+**
		- **-**
		- **/**
		- **\***
  
Пример ввода:  
```javascript
+ 10: 01 01 01 / 09          # первое число  
- 2: 0101000001/01010001     # второе число finish  
finish
```
## Формат вывода  
  
После выполнения всех операций, программа должна вывести результат каждой операции для каждой из указанных систем счисления. Вывод числа будет выполнен в соответствии с системой счисления, выбранной пользователем. Результат будет представлен в каноническом виде (несократимой дроби), с учётом знака.  
  
Пример вывода:  
```javascript
10   04 00 06 / 03 09 
16   01 09 06 / 02 07 
8    06 02 06 / 04 07 
2    01 01 00 00 01 00 01 01 00 / 01 00 00 01 01 01
```
  
## Параметры командной строки  
  
Программа принимает параметры при запуске:  
  
1. **Система счисления**: Программа должна поддерживать вывод в нескольких системах счисления, от 2 до 256.  
Пример запуска:  `./rational_numbers_calculator 10 16 2 8`
